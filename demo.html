• Open a text edtior which allows "rich text" (I used TextEdit on the mac in its Rich Text mode).
  • Open a text edtior which does not allow "rich text" (I also used TextEdit, with Rich Text turned off in the window)).
  • Open this page in your favorite browser.
  • Select as much text as you want in the browser. Make sure the selection spans lines/rows/
  • On the mac, at least, press and hold on the selected text for a moment, and then drag
  • Notice the drag feedback as you do this includes the text that is "user-select: none" even though it was not visibly selected.
  • Drop the text in a rich text editor
  • Notice that the dropped text includes the user-select: none text
  • Alternately, copy and paste the text (cmd/ctrl+C) and paste it into that editor
  • Notice that the user-select: none text is pasted
  • Drag text again and drop the text in a plain text editor
  • Notice that the dropped text does NOT include the user-select: none text
  • Alternately, copy and paste the text (cmd/ctrl+C) and paste it into that editor
  • Notice that the user-select: none text is also NOT pasted
